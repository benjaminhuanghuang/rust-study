## Install
> https://www.rust-lang.org/tools/install

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

source ~/.cargo/env
rustc --version

```

Rust is installed and managed by the `rustup`
```
  rustup 

  rustup update
```

Check version
```
  rustc -- version
```
