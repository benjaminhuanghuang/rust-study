# Hashmap in Rust

## Iterate

```rust
for (person, age) in people.iter () {
  println! ("person = {:?}, age = {:?}", person, age);
}

for person in people.keys () {
  println! ("person = {:?}", person);
}

for age in people. values () {
  println! ("age = {:?]", age);
}
```
