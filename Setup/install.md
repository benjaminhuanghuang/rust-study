## Install
> https://www.rust-lang.org/tools/install

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

source ~/.cargo/env
rustc --version

```

Check version

```
  rustc -- version
```

Update

```
  rsutup update
```

Uninstall

```
  rustup self uninstall
```
