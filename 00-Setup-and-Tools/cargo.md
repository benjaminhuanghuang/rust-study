# Cargo (npm of rust)

```sh
cargo new <proj>
```

```sh
  cargo build   (create ./target)
  cargo build --release

  cargo run     (build and run)

  cargo check
```

## Cargo

cargo is Rust build tool, dependency manager, test runner and project bootstrapper

```sh
  cargo --version
```

Create new project

```sh
  cargo new hello-world
```

Build

```sh
  cargo build        # debug version
```

Run

```sh
  cargo run
```

Test

```sh
  cargo test
```
